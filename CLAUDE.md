## プロジェクト概要

Node.js学習用のリポジトリ。非同期処理などのJavaScript/Node.jsの概念を学ぶためのサンプルコードを含む。

## コマンド

```bash
# HTTPサーバー起動
node app.js

# setTimeoutサンプル実行（番号1-6を指定）
node setTimeout_sample.js [番号]
```

## ファイル構成

- `app.js` - シンプルなHTTPサーバー（ポート3000）
- `setTimeout_sample.js` - setTimeout学習用サンプル集（番号指定で個別実行可能）
